#计算器实现说明

###功能###
	- 模拟PC计算器
	- 实现：
		加、减、乘、除；
		求方根、求平方、求三次方、求反（正负运算，可连续求反）；
		连续计算、历史表达式存储；
		退格、清空。
	
###操作说明###
	- CE  --  清除历史纪录框；
	- C   --  清除计算框；
	- ←   --  退格；
	- 若连续点击运算符号，无效；
	- 点击“=”求得结果后，可以继续点击运算符，对得到的结果进行计算
	
###实现方式说明###
    模拟PC计算器标准模式，对用户输入的表达式实时计算结果。
